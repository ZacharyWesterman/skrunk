Issue Tracker

Report Bug / Request Feature

If you're reporting a bug or other issue, please include the following in your description:
  • Where you encountered the bug / issue.
  • What actions seem to trigger it.
  • Whether it happens randomly or consistently.
  • The behavior you expected to see.
  • The actual behavior you are seeing.
If you're requesting a new feature or a change to an existing feature, please be descriptive and clear with what you want. E.g. prefer to write requests like:
The UI on [page name] feels clunky to use, I have to go through a bunch of options just to find [feature name]. I use it all the time so I'd like some sort of shortcut or streamlined way to use it.
Instead of requests like:
I don't like how [page name] feels, I can never find the option I want.
