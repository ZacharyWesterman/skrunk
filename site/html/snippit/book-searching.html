There are more sorting and filtering options than just the default keyword search.
You can click the button to see more fields which can help you narrow down your search.
  • Owned by
    • Select a user to filter to only books owned by them.
  • Borrowed
    • If checked, show only books that are currently borrowed.
    • If un-checked, show only books that are not borrowed.
    • If indeterminate, show both.
  • Keywords/ISBN
    • If input is a 10 or 13 digit number, it's assumed to be an ISBN.
    • Otherwise, keywords are searched for the most likely matches. These keywords may be found in any of the Author, Category, Title, Sub-title, or Description fields.
  • Author
    • Filter by author name. If you just type something like Lewis, it will match all authors whose name contains that text, however more complex queries can be made by typing in a regular expression . For example, to search for an author whose last name is "Lewis", you would use \bLewis$.
  • Category
    • Filter by category. If you just type something like Fiction, it will match all categories containing that text, however more complex queries can be made by typing in a regular expression . For example, to search for a category that is fiction but not juvenile fiction, you would use (?<!Juvenile )Fiction.
